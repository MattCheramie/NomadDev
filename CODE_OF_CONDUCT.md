# Code of Conduct

This project adopts the
[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. Please read the linked document for the full
expectations, enforcement guidelines, and reporting mechanics.

## Reporting

Conduct concerns go to the repository's private channel: open a
[GitHub Security Advisory](https://github.com/MattCheramie/NomadDev/security/advisories/new)
with the body "code-of-conduct report" and a maintainer will reach
out within 72 hours. The Advisory form is also the right channel for
security issues — see [`SECURITY.md`](./SECURITY.md).

If you can't use the Advisory flow, open a public issue titled
"code-of-conduct: please contact" with **no other details**, and a
maintainer will move the discussion to a private channel.

## Scope

The Code of Conduct applies to all project spaces: issues, pull
requests, the wiki, release notes, commit messages, and any
maintainer-hosted chat. It also applies when individuals
represent the project in public spaces.
